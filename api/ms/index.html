ms

Provides functions for converting between milliseconds and strings.

local ms = require(packages.ms)

ms.toNumber("1m 30s") -- 90000
ms.toString(90000) -- "1m 30s"

ms also employs some metatable magic to allow for calling the module as a function. ms will automatically determine whether to call ms.toNumber or ms.toString based on the type of the first argument. 

ms("1m 30s") -- 90000
ms(90000) -- "1m 30s"
ms(ms("1m 30s")) -- "1m 30s"

ms("1d", "s") -- 86400
ms(90000, "long") -- "1 minute, 30 seconds"

Functions​

toNumber​

</>
ms.toNumber(
value: string,
unit: Unit?
) → number

Converts a string to a number. The output number is in milliseconds unless a unit is specified. 

ms.toNumber("1m 30s") -- 90000
ms.toNumber("1m 30s", "s") -- 90

toString​

</>
ms.toString(
value: number,
format: "long" | "short"?,
inputUnit: Unit?
) → string

Converts a number to a string. The number is assumed to be in milliseconds. unless an inputUnit is specified. Values are rounded to the nearest whole number. Weeks are not displayed. Milliseconds are not displayed unless the input value is less than 1 second. 

ms.toString(9000) -- "9s"
ms.toString(9001) -- "9s"
ms.toString(90, "long") -- "90 milliseconds"

ms.toString(9, nil, "s") -- "9s"
ms.toString(9, "long", "s") -- "9 seconds"
